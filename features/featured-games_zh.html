核心功能專題

特色遊戲支持

GTA、荒野大鏢客、紙嫁衣等重點遊戲支持與排查入口

功能概覽

特色遊戲支持頁用於集中說明高頻遊戲的支持狀態、常見啟動問題和對應的 FAQ/遊戲庫入口。

適合場景

特色遊戲入口

適合 GTA、荒野大鏢客、紙嫁衣等需要單獨說明啟動方式和注意事項的遊戲。

特殊啟動參數

部分遊戲需要特定參數、組件或網絡狀態，頁面幫助用戶先確認這些差異。

FAQ 分流

把常見特色遊戲問題從通用 FAQ 中拆出，減少用戶在長頁面裡查找的時間。

可以解決什麼

  • 把高頻遊戲的支持說明集中在一個入口。
  • 引導用戶從遊戲庫更新頁確認支持狀態。
  • 把常見啟動、目錄、版本和網絡問題連接到 FAQ。

推薦使用流程

1
先在遊戲庫更新頁確認當前遊戲是否已收錄。
2
根據遊戲名稱或 AppID 找到對應說明。
3
按照 README 或 FAQ 的通用流程啟動和排查。
4
遇到遊戲專屬問題時查看補丁插件頁或反饋給作者。

注意事項

  • 特色遊戲支持會隨版本更新變化。
  • 不同遊戲的啟動器、第三方平臺和網絡要求不同。
  • 建議先確認遊戲版本和 AppID。

常見疑問

特色遊戲為什麼要單獨列出來？

這些遊戲的啟動方式、組件依賴或網絡驗證更容易和普通遊戲不同，單獨說明更清楚。

列表裡沒有的遊戲怎麼辦？

先查遊戲庫更新和 FAQ，如果暫未支持，可以等待後續更新或關注公告。
