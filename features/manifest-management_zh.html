核心功能專題

Steam 遊戲清單管理

作者統一維護清單，客戶端倒序展示，一鍵入庫後直接顯示到 Steam 遊戲庫

功能概覽

LuckyGameTools 的遊戲清單由作者在客戶端統一添加、更新和維護，用戶不需要自己維護清單文件。客戶端會按加入時間倒序平鋪展示已支持遊戲，點擊“一鍵入庫”後，軟件會自動下載託管在 GitHub 上的清單文件並導入到本機 Steam 庫；導入完成後無需重啟 Steam，遊戲會直接在 Steam 遊戲庫中顯示並正常使用。

適合場景

Steam 入庫整理

適合想直接從客戶端列表裡找到遊戲並入庫的用戶，不需要手動查找或維護清單文件。

作者統一維護

清單由作者在客戶端統一添加、更新和維護，用戶只需要保持客戶端列表可正常刷新。

按加入時間倒序展示

客戶端會把已加入的遊戲按時間倒序平鋪出來，最新支持的遊戲更容易被看到。

可以解決什麼

  • 減少用戶手動找清單、拖清單、維護清單版本的成本。
  • 客戶端按加入時間倒序平鋪展示，最新支持遊戲更容易找到。
  • 點擊一鍵入庫後自動從 GitHub 下載清單文件，完成後直接在 Steam 遊戲庫顯示。

推薦使用流程

1
打開客戶端，在按加入時間倒序排列的遊戲清單中找到目標遊戲。
2
點擊目標遊戲的一鍵入庫按鈕。
3
軟件會自動下載 GitHub 上維護的清單文件並完成導入。
4
無需重啟 Steam，回到 Steam 遊戲庫即可看到遊戲並正常使用。

注意事項

  • 清單由作者統一添加、更新和維護，用戶不需要自行維護清單文件。
  • 一鍵入庫需要客戶端能夠正常聯網，以便下載 GitHub 上的清單文件。
  • 如果入庫後未顯示，優先檢查網絡下載是否成功、遊戲是否已被客戶端支持，以及 Steam 是否處於正常登錄狀態。

常見疑問

清單需要用戶手動維護嗎？

不需要。清單由作者在客戶端統一添加、更新和維護，用戶只需要在客戶端裡找到遊戲並點擊一鍵入庫。

一鍵入庫後需要重啟 Steam 嗎？

不需要。點擊一鍵入庫後，軟件會下載 GitHub 上的清單文件並導入，本機 Steam 遊戲庫會直接顯示並可以正常使用。
